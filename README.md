# Parking Lot (using Node JS)

## Quick Start

### Play the Game

```bash
# Run install/download depedencies  & unit testing
$ bin/setup
```

```bash
# Run app
$ bin/parking_lot
```

```bash
# Run app with file input
$ bin/parking_lot file_input.txt
```

```bash
# run functional spec
$ bin/run_functional_tests
```
